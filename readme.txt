Загрузите плагин ark_commenteditor на свой блог. Download the ark_commenteditor plugin to your blog.2. Активируйте его. Activate it.3. Настройте по своему усмотрению в панели управления на странице `Настройки - ARK WYSIWYG Comment Editor`. Set at its discretion in the control panel on page `Settings - ARK WYSIWYG Comment Editor`.4. Более подробную информацию о плагине Вы можете найти на [странице плагина](http://blog.ddw.kz/plagin-ark-wysiwyg-comment-editor-vizualnyj-redaktor-kommentariev.html). For more information about the plugin can be found at [plugin page](http://blog.ddw.kz/plagin-ark-wysiwyg-comment-editor-vizualnyj-redaktor-kommentariev.html).== Screenshots ==1. Плагин корректно работает с древовидными комментариями.= Будет-ли плагин работать, если у меня свой шаблон и форма комментариев уже встроена в него? =В этом случае плагин может не сработать, так как его действие основано на перехвате вызова функции comment_form().== Changelog === 1.0 =* Первая версия== Upgrade Notice === 1.0 =* Первая версия
